Lộ trình học

1. Chuẩn bị môi trường

Cài đặt JDK 17+ và một IDE như IntelliJ IDEA hoặc VS Code (Java Extension Pack). Thiết lập biến môi trường JAVA_HOME và kiểm tra java -version.

2. Cơ bản ngôn ngữ

Làm quen cú pháp: biến, kiểu dữ liệu, toán tử, nhập/xuất, điều kiện (if/else), vòng lặp (for/while), mảng và phương thức.

3. Hướng đối tượng

Hiểu lớp, đối tượng, tính đóng gói, kế thừa, đa hình, interface và trừu tượng. Viết mã sạch, đặt tên có ý nghĩa.

4. Xử lý ngoại lệ & Collections

Sử dụng try/catch, ném ngoại lệ, các cấu trúc List, Set, Map và duyệt dữ liệu hiệu quả.

5. I/O & Thực hành

Đọc/ghi file cơ bản, tổ chức project, viết test đơn giản, làm mini project để tổng hợp kiến thức.

Bài học nhanh

1) Hello World 
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}
  • class: khuôn mẫu tạo đối tượng
  • main: điểm vào chương trình
  • System.out.println: in ra màn hình
2) Biến & Kiểu dữ liệu 
int age = 20;          // số nguyên
double pi = 3.14;     // số thực
boolean valid = true; // logic
char grade = 'A';     // ký tự
String name = "An";   // chuỗi

Đặt tên biến rõ nghĩa, dùng camelCase.

3) Điều kiện & Vòng lặp 
int score = 75;
if (score >= 90) {
  System.out.println("A");
} else if (score >= 80) {
  System.out.println("B");
} else {
  System.out.println("C hoặc thấp hơn");
}

for (int i = 0; i < 3; i++) {
  System.out.println(i);
}
4) Mảng & Phương thức 
int[] arr = {1, 2, 3};
for (int x : arr) {
  System.out.println(x);
}

static int add(int a, int b) {
  return a + b;
}
// Gọi: int c = add(2, 3);
5) OOP cơ bản 
class Person {
  private String name;

  public Person(String name) {
    this.name = name;
  }

  public String getName() {
    return name;
  }
}

class Student extends Person {
  private int id;

  public Student(String name, int id) {
    super(name);
    this.id = id;
  }
}

Bài tập thực hành

  1. Viết chương trình nhập vào một số nguyên n và in ra tổng 1 + 2 + ... + n.
  2. Viết phương thức kiểm tra số nguyên tố. Dùng trong chương trình để in các số nguyên tố nhỏ hơn 100.
  3. Tạo lớp Book gồm title, author, price; viết getter/setter và hàm in thông tin.
Gợi ý lời giải (mở khi cần) 
// 1) Tổng 1..n
int sum = 0;
for (int i = 1; i <= n; i++) sum += i;

// 2) Kiểm tra số nguyên tố
static boolean isPrime(int x) {
  if (x < 2) return false;
  for (int i = 2; i * i <= x; i++) if (x % i == 0) return false;
  return true;
}
for (int i = 2; i < 100; i++) if (isPrime(i)) System.out.print(i + " ");
